## Introduction
The Library App is designed to help users manage their book collections, keep track of borrowed books, and find books easily within a mobile interface. This application is ideal for both personal and small organizational use.

## Features
- **User Authentication:** Secure login and sign-up feature to protect user data.
- **Book Management:** Add, remove, and manage book listings.
- **Borrowing System:** Keep track of borrowed books and due dates.
- **Search Functionality:** Easily search for books by title, author, or genre.
- **Offline Support:** Works offline using SQLite as a local database.
- **Notifications:** Reminders for due books.

## Installation
To get started with the Library App, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/JameTNK404/LibraryApp-45002-thinnakrit.git
   ```
2. Navigate to the project directory:
   ```bash
   cd LibraryApp-45002-thinnakrit
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   expo start
   ```

## Usage
After starting the application, you can create an account or log in using existing credentials. Once logged in, you can add and manage your book collection and track borrowed books.

## Technologies Used
- **React Native:** For building the mobile application.
- **Expo:** For streamlined development and debugging.
- **SQLite:** For local data storage.
